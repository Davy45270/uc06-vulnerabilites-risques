# UC06 Vulnérabilités & Risques

## Expression de besoin fonctionnelle
- **Objectifs** : recenser vulnérabilités/risques, évaluer impact/probabilité, suivre remédiations.
- **Acteurs** : CISO, Risk managers, équipes techniques.
- **Flux clés** : détection → classification → scoring → plan d’action → suivi → clôture.
- **Exceptions** : risques critiques escaladés; acceptation de risque formalisée.
- **Critères d’acceptation** : registre central, workflow de validation, reporting par criticité.

## Analyse ROI
- **Gains** : réduction exposition, priorisation claire, conformité renforcée.
- **Coûts** : intégration sources (scanners), gouvernance sécurité.
- **KPI cibles** : temps moyen de remédiation, nombre de vulnérabilités critiques ouvertes, taux de conformité.
